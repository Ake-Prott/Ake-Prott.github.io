Robot Shop
Descripción

Electro Shop es un proyecto académico desarrollado como un e-commerce funcional orientado a la gestión de productos electrónicos.

El objetivo principal fue diseñar y desarrollar una página web conectada a una base de datos que permitiera realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar), aplicando los conocimientos de desarrollo web, bases de datos y conexión backend.

La aplicación permite la gestión de usuarios, catálogo de productos, carrito de compras e historial, integrando frontend y backend en un entorno funcional.

Tecnologías utilizadas
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: PHP
  • Base de datos: MySQL
  • Control de estructura: Archivos organizados por módulos (login, catálogo, carrito, etc.)
Mi Rol en el Proyecto
  • Creación y diseño de la base de datos.
  • Desarrollo del frontend general.
  • Desarrollo del backend en PHP.
  • Implementación de la conexión a la base de datos.
  • Desarrollo de las operaciones CRUD.
  • Elaboración de la documentación técnica del proyecto.

App de Gestión de Tareas
Descripción

Proyecto académico enfocado en el diseño de la interfaz web para RebuildWare, una empresa especializada en mantenimiento preventivo y correctivo de equipos de cómputo.

El proyecto consistió en la investigación, planificación y diseño de la experiencia de usuario, desarrollando prototipos en Figma basados en principios de usabilidad, leyes heurísticas y fundamentos UX.

Se diseñaron las secciones principales del sitio web, incluyendo la estructura general de la página y los apartados de servicios, adaptando la interfaz para distintos dispositivos (desktop, laptop, tablet y móvil).

El objetivo fue diseñar una interfaz web funcional y centrada en el usuario que representara los servicios de la empresa, aplicando principios de usabilidad, jerarquía visual y experiencia de usuario para mejorar la claridad y navegación del sitio.

Tecnologías utilizadas y herramientas utilizdas
  • Figma (diseño de interfaces y prototipado)
  • Investigación UX
  • Leyes heurísticas de usabilidad
  • Principios de experiencia de usuario (UX)
  • Diseño responsive (adaptación a múltiples dispositivos)
Mi Rol en el Proyecto
  • Investigación sobre principios UX y leyes heurísticas.
  • Participación en la definición de las secciones del sitio web.
  • Diseño de las secciones correspondientes a Mantenimiento en Software.
  • Diseño adaptable para distintos dispositivos (responsive).

App de Gestión de Tareas
Descripción

Este proyecto consiste en el desarrollo de un sistema web para el dispensario médico Salud Total, cuyo objetivo es gestionar el inventario de medicamentos, administrar proveedores y controlar el acceso de usuarios mediante autenticación con sesiones.

El sistema permite registrar medicamentos, consultar el inventario disponible, actualizar cantidades y eliminar registros, aplicando operaciones CRUD completas. Además, se implementó un sistema de inicio de sesión para restringir el acceso únicamente al personal autorizado.

Tecnologías utilizadas
  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP
  • Base de datos: MySQL
  • Comunicación: Métodos GET y POST
  • Gestión de sesiones: PHP Sessions
